USB Powered And Configurable Field Programmable Gate Array:

Configurations:

JTAG

The jtag chain is always available despite the mode settings as stated in data sheet of the FPGA DS312 on page 101. The chain consists of the CPLD, XC2C64A VQ100, and then the FPGA, XC3S250E VQ100. The SPI flash can be included on the chain via Impact, XAPP974.

Slave Parallel

DS312 pg. 95

DS_FT245R

The slave parallel mode as described in DS312 on pg. 95 shows a memory configuration source in figure 62. The memory configuration source is handled by a PC, FTDI driver, USB, FT245R chip, and the CPLD. This is a seamless transition after the CPLD is configured via the JTAG chain.

SPI Serial Flash

XAPP974

DS312 pg. 75

M25P16

Once the programming of the SPI Flash via the indirect method on the JTAG chain has been completed the FPGA can come up in a stand alone operation. Only power is required. It is most probable that a the FPGA can be configured through the USB to allow programming of the SPI flash although this has not been attempted.
